Tópico Base: usado para publicação de status e recebimento de comandos.
Exemplo: com tópico "smcr" e ID único "smcr_123abc", será usado "smcr/smcr_123abc/pin/X/state" para publicar status.

Home Assistant Auto-Discovery: quando habilitado, o módulo publicará automaticamente as configurações de descoberta no formato do Home Assistant, permitindo que seus pinos apareçam automaticamente na interface.

Discovery - Tamanho do Lote: número de entidades publicadas por vez (padrão: 4).
Discovery - Intervalo do Lote(ms): tempo de espera entre cada lote de publicação (padrão: 100ms).
Discovery - Repetir a Cada: intervalo para re-executar o discovery periodicamente em segundos (padrão: 900 = 15 minutos). Use 0 para desabilitar a repetição periódica.
