Operacoes de Reset e Limpeza do Sistema
Atencao! As operacoes desta pagina sao irreversiveis. Use com cuidado!

🔄 Reinicializar Sistema

Reinicia o ESP32 mantendo todas as configuracoes salvas. Util para resolver problemas temporarios.

🔧 Reset de Configuracoes (Preservando Rede)

Limpa todas as configuracoes EXCETO as informacoes de rede (hostname, SSID e senha WiFi). O sistema continuara conectado na mesma rede.

📌 Limpar Configuracoes de Pinos

Remove todas as configuracoes de pinos/reles. As configuracoes gerais do sistema serao mantidas.

⚠️ Reset Completo (Factory Reset)

APAGA TUDO! Remove todas as configuracoes, inclusive rede. O sistema voltara ao modo Access Point para nova configuracao.
